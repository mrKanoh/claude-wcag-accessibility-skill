## 📋 Cómo Usar Este Repositorio

1. **Referencia oficial**: Comienza con W3C WAI para especificaciones.
2. **Guías prácticas**: WebAIM, Deque, AllAccessible para implementación.
3. **Código & patrones**: APG (W3C), Inclusive Components, repositories GitHub.
4. **Auditoría**: axe, WAVE, Lighthouse, pa11y.
5. **Investigación**: Revisa la sección de papers académicos para contexto normativo y evidencia.
6. **Comunidad**: WebAIM, A11Y Project, IAAP para red y certificación.
