Accessible Carousel

WCAG 2.1/2.2 AA. Keyboard: Arrow keys navigate, Space pauses. Auto-rotation pauses on hover, focus, or with prefers-reduced-motion.

🎹

Keyboard First

Every interactive element must be reachable with keyboard alone. Test Tab, Enter, Space, and Arrow keys.

🔍

Focus Visibility

Focus indicators must have 3:1 contrast and cover ≥4 CSS px area — WCAG 2.4.11 and 2.4.13.

🎨

Color Contrast

Normal text needs 4.5:1, large text 3:1, and UI components 3:1 against adjacent backgrounds.

📱

Touch Targets

Minimum 24×24 CSS px (WCAG 2.5.8 AA) or 44×44 with no adjacent targets (WCAG 2.5.5 AAA).
